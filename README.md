# MERN stack 
MongoDB, Express, React, Redux, Node, express-session and datatable

## Install

#### Server
  $ cd server

  $ npm install

#### Client
  $ cd client

  $ npm install

## Run & Start

#### Server
  $ cd server

  $ npm start

#### Client
  $ cd client

  $ npm start
